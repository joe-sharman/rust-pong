# Pong

Pong implemented in rust.

Includes score board and timer.

Player 1 is controlled by the user, with the `w` and `s` keys. The paddle has momentum to add difficulty.

Player 2 is controlled by the computer. It uses a simple algorithm to track the ball.

![](/../main/images/demo.png?raw=true)

This project was created to practice rust - specifically polymorphism and encapsulation.

Using the [macroquad](https://macroquad.rs/) crate.

_Originally published on the 20th of September 2025_.
